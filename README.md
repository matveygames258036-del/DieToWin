# DieToWin GAME

---

## **Описание:**
Игра "**DieToWin**" это игра наоборот где нужно уиереть чтобы пройти уровень
По ходу игры вам расскажет встроенное обучение которое будет помогать новым игрокам

## **Разработка и поддержка:**
Если вы хотите денежно поддержать разработчика можете отправить копеечку на Яндекс пэй способы:
	1. Способ: **По номеру карты**:
		При переводе введите номер карты: 2204 3113 1330 4199
	2. Способ **По номеру телефона**:
		При переводе введите номер телефона +7 (913) 346-28-82
Если вы хотите улучшить игру как либо можете сделать форк на её основе

## **Минимальные системные требования:**
ОС: Windows 7 (32-бита);
Разрешение экрана: 640 X 480;
ОЗУ: 512 МБ;
ПЗУ: 128 МБ.
